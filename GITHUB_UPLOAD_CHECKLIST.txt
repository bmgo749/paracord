USER CLONE INSTRUCTIONS:
=========================
Users cloning your repo will:

1. Clone repository:
   git clone https://github.com/yourusername/paracord.git
   cd paracord

2. Install:
   pip install nextcord
   pip install -e .

3. Install extension:
   - Press F5 in VSCode (temporary)
   - OR copy paracord-vscode to extensions folder (permanent)

4. Create bot.cord and run:
   para run bot.cord
